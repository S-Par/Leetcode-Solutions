## Problem 5: Longest Palindrome Substring
- Difficulty Level: Medium 

Given a string s, return the longest palindromic substring in s.

#### Constraints:
- 1 <= s.length <= 1000
- s consist of only digits and English letters.


---
### Examples:
```
Input: s = "babad"
Output: "bab"
```

```
Input: s = "cbbd"
Output: "bb"
```

---
### Solution:
- Constructed a bottom-up Dynamic Programming solution that builds a quadratic time function
